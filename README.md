# BOM_Check工具使用方法

1、双击运行BOM_Check_ui.exe等待一会

2、打开界面如图

![图片名](https://raw.githubusercontent.com/mygithubma/BOM_Check/main/png/001.png)

3、点击加载BOM1和加载BOM2自动跳转选择自己的两份BOM

4、点击开始比对，自动对比两个BOM的位号

![](E:\Typoral\BOM_Check\002.png)

5、点击清除可以清除当前正在检测的BOM文件

![](E:\Typoral\BOM_Check\003.png)

6、点击BOM模版会自动从仓库下载该工具的模版文件，用户只需改为模版的样式即可，下载路径为工具exe文件所在路径

![](E:\Typoral\BOM_Check\004.png)

7、点击使用说明右侧显示模版所需要求

![](E:\Typoral\BOM_Check\005.png)

8、点击关闭结束运行
